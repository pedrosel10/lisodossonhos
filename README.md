# Liso dos Sonhos - Link in Bio

Landing page premium de links para o salão Liso dos Sonhos, desenvolvida com React, Vite e Tailwind CSS.

## Tecnologias Utilizadas

- **React**: Biblioteca principal para a interface.
- **Vite**: Build tool extremamente rápida.
- **Tailwind CSS v4**: Para estilização moderna e eficiente.
- **Lucide React**: Biblioteca de ícones.

## Como rodar localmente

1. Clone o repositório:
   ```bash
   git clone https://github.com/pedrosel10/lisodossonhos.git
   ```
2. Instale as dependências:
   ```bash
   npm install
   ```
3. Inicie o servidor de desenvolvimento:
   ```bash
   npm run dev
   ```

## Funcionalidades

- Design responsivo e premium.
- Animações de entrada elegantes.
- Cursor customizado (desktop).
- Efeito de iluminação dinâmica seguindo o mouse.
